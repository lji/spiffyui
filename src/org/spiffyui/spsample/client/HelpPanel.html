Get Help With Spiffy UI

We're working on making the Spiffy UI framework open source, but for now it is available only to Novell projects. You can use the Spiffy UI framework for your project. We support new projects and all of the open source code included in this project has been reviewed by the Novell open source review board.

Try out the sample application

Downloading and trying out this entire website, built on the Spiffy UI framework, is a fast and easy way to get plenty of sample code and documentation. You can check it out anonymously and build it in about a minute. It deploys to any J2EE application server you want.

1. Use the following command to check out the Spiffy Sample application which also includes the Spiffy UI framework: 

    svn co https://svn.provo.novell.com/svn/spiffyui/sample spsample

2. Set the environment variable JAVA_HOME to the location of your JDK (1.6 or higher).

3. Place the external/ant/bin directory in your path (optional).

4. Execute this command line from the spsample directory: 

    ant (if not on your path then: external/ant/bin/ant)

5. Once the build completes you can deploy the spsample.war file from dist/spsample.war.
