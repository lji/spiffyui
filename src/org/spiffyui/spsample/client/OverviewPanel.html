Spiffy UI/UX Framework

Spiffy UI framework: beautiful, fast, flexible, secure, maintainable applications with GWT and REST

The Spiffy UI framework takes the power of GWT and adds patterns, widgets, and utilities to make beautiful, fast, secure, maintainable web applications. Spiffy UI integrates JQuery and other popular JavaScript technologies into a reusable framework that works like a standard GWT application. It adds a REST framework with security and flexible CSS to create applications that feel good.

Flexibilty

Spiffy UI is flexible and modular in its design. Each piece is meant to work alone. Use only the pieces you want and leave the rest.

Ext GWT, Smart GWT, and many other GWT frameworks integrate well with the flexible design of the Spiffy UI framework. The framework also makes it easy to integrate JavaScript libraries. Many of the Spiffy UI functions are available in JavaScript.

Spiffy philosophy

The Spiffy UI framework started as part of the Novell Identity Manager Reporting Module and was later extracted into a reusable framework. The philosophy of the framework is best explained in the Spiffy UI video.

The rules we follow for the UI of this framework are explained in Fluid Grids, Vertical Rhythm, and CSS Blocking.

Background on the mechanism we use for calling REST APIs from JavaScript and GWT is discussed in Calling REST from GWT with a little bit of JQuery.

Why GWT

GWT is easy to use, and you get all the advantages that come with coding in Java. It is type safe. You can enforce coding style with checkstyle. You can run Sonar and other static code analysis tools against it to keep your code maintainable. Custom reusable widgets can be developed quickly and wrapping existing pure JavaScript widgets can also be quite simple.

Included libraries

While enjoying the benefits of GWT, we also enhance and fill some gaps by incorporating some of the best open source libraries available:

  • Apache Commons Codec version 1.4
  • Apache HTTP Client version 4.1
  • Base64 encode / decode
  • DateJS version 1.0 alpha
  • dshistory
  • GlassFish Jersey
  • Humanized Messages
  • jQuery version 1.4.2
  • jQuery Time Picker version 0.2
  • jQuery UI version 1.8
  • reset.css version 2.8.0r4
  • Sliding Grid
