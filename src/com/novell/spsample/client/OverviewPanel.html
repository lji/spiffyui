Spiffy UI Framework

Spiffy UI framework: beautiful, fast, secure, maintainable applications with GWT and REST

The Spiffy UI framework takes the power of GWT and adds patterns widgets and utilites for making beautiful, fast, secure, maintainable web applications. Spiffy UI integrates JQuery and other popular JavaScript technologies into a reusable framwork that works like standard GWT application. It combines a REST framework with security and flexible CSS to create applications that feel good.

The Hard Sell

Spiffy UI gives you:

  • Framework and patterns for calling REST with JSON
  • REST error handling framework
  • Authentication and security UI and infrastructure
  • CSS templates and reusable styles
  • Date handling and localization
  • Form layouts
  • Reusable widgets
  • Build integration with JavaScript and CSS compression

Spiffy philosophy

The Spiffy UI framework started as part of the Identity Manager Reporting Module and was later extracted into a reusable framework. The philosophy of the framework is best explained in the Spiffy UI video.

The rules we follow in this framework are explained in Fluid Grids, Vertical Rhythm, and CSS Blocking.

The mechanism we use for calling REST APIs from JavaScript and GWT is discussed in Calling REST from GWT with a little bit of JQuery.

The sliding grid panel in this framework is described in Create Your Own Sliding Resizable Grid.

The multi-valued auto-complete control in this framework is explained in detail in Creating a Multi-Valued Auto-Complete Field Using GWT SuggestBox and REST.
