Spiffy UI/UX Framework

Spiffy UI framework: beautiful, fast, flexible, secure, maintainable applications with GWT and REST

The Spiffy UI framework takes the power of GWT and adds patterns, widgets and utilities for making beautiful, fast, secure, maintainable web applications. Spiffy UI integrates JQuery and other popular JavaScript technologies into a reusable framework that works like a standard GWT application. It combines a REST framework with security and flexible CSS to create applications that feel good.

Flexibilty

This framework is flexible and modular in its design. Use only the pieces you want and leave the rest. Spiffy UI also combines well with other GWT frameworks and promotes the integration of JavaScript frameworks.

Spiffy philosophy

The Spiffy UI framework started as part of the Novell Identity Manager Reporting Module and was later extracted into a reusable framework. The philosophy of the framework is best explained in the Spiffy UI video.

The rules we follow for the UI of this framework are explained in Fluid Grids, Vertical Rhythm, and CSS Blocking.

Background on the mechanism we use for calling REST APIs from JavaScript and GWT is discussed in Calling REST from GWT with a little bit of JQuery.

Why GWT

GWT is easy to use, and you get all the advantages that come with coding in Java. It is type safe. You can enforce coding style with checkstyle. You can run sonar statistics against it. You can keep your code very maintainable. Custom reusable widgets can be developed quickly and wrapping existing pure JavaScript widgets can also be quite simple.
