Get Help With Spiffy UI

We're working on making the Spiffy UI framework open source, but for now it is available only to Novell projects. You can use the Spiffy UI framework for your project. We support new projects and all of the open source code included in this project has been reviewed by the Novell open source review board.

Try it out

Trying the Spiffy UI framework is easy. You can check it out anonymously and build it in about a minute. It deploys to any J2EE application server you want.

Get all the information you need to try the framework on the Spiffy UI Build Setup page.

If you have more questions about the framework or want to talk about how to use it in your project please contact Zack Grossbart zgrossbart@novell.com.

Included libraries

The Spiffy UI framework includes code from the following open source projects:

  • Base64 encode / decode
  • DateJS version 1.0 alpha
  • dshistory
  • Humanized Messages
  • jQuery version 1.4.2
  • jQuery Time Picker version 0.2
  • jQuery UI version 1.8
  • reset.css version 2.8.0r4
  • Sliding Grid
