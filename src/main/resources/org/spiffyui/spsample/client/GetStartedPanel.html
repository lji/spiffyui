Getting Started With Spiffy UI

The Spiffy UI framework is included as a subproject and referenced in your GWT project. You can start using the framework in four simple steps.

Requirements

To run the Spiffy UI framework you'll need:

  • Java 1.6 or higher
  • GWT 2.0 or higher
  • A J2EE application server (only if you need the server-side parts)

1. Include spiffy in your project

The SpiffyUI framework is typically included as a Subversion external reference: https://svn.provo.novell.com/svn/spiffyui/trunk.

Include it in your project by setting the svn:externals property on a directory in your project. You can set the property using UI tools or with this command line: 

    svn propset svn:externals "spiffyui https://svn.provo.novell.com/svn/spiffyui/trunk" <directory>

After you set the property, check in and then update your working copy. The Spiffy UI framework will be in the directory you chose. In the sample project the Spiffy UI framework is in the external directory.

2. Build spiffy

The Spiffy UI framework uses an Apache Ant build script to compile and compress JavaScript and CSS files using the Google Closure Compiler and the YUI Compressor. You must include Ant in your build environment. For projects within Novell, you may reference the Spiffy UI Build Setup page. You can invoke the Spiffy UI build from your own Ant build.xml file with the ant task. Like this: 

    <ant antfile="${build.base}/external/spiffyui/build/build.xml"
         dir="${build.base}/external/spiffyui">
       <property name="spiffy.www" value="${root}/bin/www"/>
       <property name="spiffy.classes" value="${root}/bin/classes"/>
    </ant>

In the sample we've placed the Spiffy UI framework in the external/spiffyui folder of our project. The spiffy.www property specifies the location to generate the Spiffy UI framework CSS, JavaScript, and image files. The spiffy.classes specifies the location of the class Spiffy UI class files. These class files are a small number of servlets to support the framework.

3. Reference spiffy in HTML

Now the CSS and JavaScript files are included in your project when you build. The next step is to reference them in your HTML file. The Spiffy UI framework includes many JavaScript and CSS files, but they are all combined into two files for faster application loading. Reference these three files and the JQuery library in the head section of your HTML files like this: 

    <head>
    <script type="text/javascript" src="jquery-1.4.3.min.js"></script>
    <script type="text/javascript" src="spiffyui.min.js"></script>
    …
    </head>

4. Import spiffy in GWT module

The last step is to import the Spiffy UI framework GWT module. Add the following line to your GWT module file: 

    <inherits name="org.spiffyui.spiffyui" />

That's all it takes.
