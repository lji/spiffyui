Build the full Spiffy UI Application

When you're ready to take the next steps you can build this entire website and get an inside look at the more advanced Spiffy UI application. Check it out and build it in about a minute.

1. Use the following command to check out the Spiffy UI application which also includes the Spiffy UI framework: 

    svn co http://spiffyui.googlecode.com/svn/trunk spiffyui

2. Add the external/ant/bin directory to your path. You can also use your own version of Ant 1.8 with Apache Ivy.

3. Execute this command line from the spiffyui directory: 

    ant all run

This will build the project and launch it with an embedded Jetty server. You can also deploy the spsample.war file to your favorite J2EE server.

Note: The first time you build the project it must download project dependencies. After that the build process should take about one minute to complete.

Why's it called Spiffy

The Spiffy UI framework started as part of the Novell Identity Manager Reporting Module. The Reporting Module spent a long time without an official name so we started calling it Our Spiffy Reporting Tool. The name just stuck.
