Get Involved

Spiffy UI is an open source project with a supportive community. We welcome coders, testers, documenters, and anyone else with ideas, suggestions, feedback, or any type of help for the project.

The best place to start is our Google Code page, where you'll find source code, groups, and information about contributing to the project.

Give us feedback

The easiest way to get involved is to give us some feedback. Have a great idea for a new feature? A cool way of getting something done? Let us know about it on the Spiffy UI Google Group.

Share the framework

Another great way to help out is to talk about us on Twitter, Facebook, and any other social sharing service. The more people who know about Spiffy UI, the stronger our community will become.

For special bonus points, use the #spiffyui hashtag when you talk about us on Twitter.

Write an article

If you like Spiffy UI and use it in your project, we would love to hear about it. Write an article about Spiffy UI and we'll link to it from our project home page.

Report bugs

If you find bugs, we want to hear about them and fix them. Please file a bug report on our issue tracker and let us know what you find.

Not sure if something is a bug? You can ask about it on the Spiffy UI Google Group.

Contribute

Can you sling code like a pro? Help us out with Spiffy UI. We're always happy to take patches, bug fixes, and new features.

Take a look at the Spiffy UI Community Guide to see the processes for editing and submitting code to Spiffy UI. We also keep a list of Spiffy UI project for new users.
