Who uses Spiffy UI?

The Spiffy UI framework started as a part of Novell Identity Manager. Spiffy is the foundation of Identity Manager, the Novell Compliance Management Platform, and the upcoming Novel Sentinel 7.

Tens of thousands of people use Spiffy UI every day.

Supported browsers

The Spiffy UI framework supports:

  • Firefox 2, 3, and 4
  • Google Chrome
  • Internet Explorer 7, 8, and 9
  • Safari 4 and 5
  • Opera 10 and 11
  • Dolphin 4.5

Supported servers

Spiffy UI is a Java framework, but it doesn't require Java. You can run it on any web server. We really mean it. If it can serve files then it can host Spiffy UI.

Spiffy UI applications are running on all the major Java J2EE servers as well as servers running PHP, Erlang, and node.js.

Flexibilty

Spiffy UI is flexible and modular in its design. Each piece works alone. Use only the pieces you want and leave the rest.

Ext GWT, Smart GWT, and many other GWT frameworks integrate well with the flexible design of the Spiffy UI framework. The framework also makes it easy to integrate JavaScript libraries. Many of the Spiffy UI functions are available in JavaScript.

Mobile

The flexible nature of this framework makes it easy to support smart phones and tablets. The application will automatically resize and change the navigation based on the size and type of device being used. Try it on your browser by making your window width less than 800 pixels or open this application on your favorite mobile device.

Included libraries

We love GWT. We love JavaScript. We love putting them together.

Spiffy UI includes JQuery and JQuery UI. They work well with JQuery and fill in a few of the gaps. We also include a few other libraries to fill in the gaps and create awesome applications:

  • jQuery
  • jQuery UI

  • Apache Commons Codec
  • Apache HTTP Client
  • Base64 encode / decode
  • DateJS version 1.0 alpha
  • dshistory
  • Humanized Messages
  • jQuery Time Picker
  • JSON.org
  • reset.css
  • Sliding Grid

  • This sample application also includes JQuery/QUnit for unit tests, but it isn't part of the framework.

All of these libraries and the Spiffy UI framework code compress down into a small download footprint for your application. You can also pick and choose just the pieces you need for even faster page loading.
